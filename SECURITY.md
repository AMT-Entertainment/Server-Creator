# Security

## Reporting a Vulnerability

Please report security issues to AMT Entertainment via GitHub Issues. Do not disclose vulnerabilities publicly until they have been addressed.

## Best Practices

- Keep dependencies updated (`npm audit` / `npm run audit`)
- The app runs locally on your machine — no external servers are contacted except for API calls to PaperMC, CurseForge, Modrinth, and GitHub release endpoints
- Server data and configurations never leave your machine
